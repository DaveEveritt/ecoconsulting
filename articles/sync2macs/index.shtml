Sync two macs

Dave Everitt, last updated

Just some links from the process of researching this not-a-picnic topic a few years ago.

If you're not using Dropbox (or similar like Microsoft's) or don't have enough cloud storage, and aren't using Apple's cloud (you might be on an older system that doesn't support it) these links might be of use.
